Getting Started: Create Your First NetBeans Project

In order to start developing your first web-page, you need to install NetBeans, a powerful multi-platform IDE.

You can download NetBeans through the following link:

https://netbeans.org/downloads/


If you need assistance with the installation process, contact us or scroll down to view our tutorial.





Step 1: Download

Since we are going to focus on web development, you must download the version of NetBeans that supports HTML5 and PHP technologies

Therefore, we recommend the full version.

It is the rightmost option, labeled "All"



Step 2: Install

Open the downloaded package to install. Reference this guide for assistance:

Once you are finished with this step, you should be able to open the NetBeans IDE

A console will pop up with a start page. Next, you need to create a NetBeans Project.:



Step 3: Project Setup

This is the final step! We are almost there!!!


To create a new project, do the following:

1. Click on the File tab on the top of the screen and select the first option - New Project

Alternatively, you can click the second icon to the left of the top-left corner of the screen

2. Then, select the type of project you want to create. You are just getting started with Web Development, so choose HTML5

The project is automatically saved in a directory on your computer. This folder is likely called "NetBeansProjects"

3. An index.html file will be automatically created. This is your main file. It is the "Start Page." You will learn more about this later.

For now, familiarize yourself with the layout and design of the NetBeans IDE.


NOW YOU CAN START YOUR CODING LESSONS !!!
